AdminTools

A Bukkit / Spigot admin toolkit with a multi-page inventory editor.

download

Intro

AdminTools is a server-side plugin for Bukkit / Spigot / Paper that provides quality-of-life tools for server administrators:

  • Multi-page inventory editor (main inventory, armor, offhand, ender chest)
  • Live write-back syncing to the target player
  • Shift-click and hotbar protection to prevent item loss
  • Vanish toggle for staff moderation
  • Fly toggle for quick navigation

Multi-page inventory GUI

Open a player’s inventory with //admin search <player> and navigate between:

  • Page 1: Main inventory
  • Page 2: Armor & offhand (not working)
  • Page 3: Ender chest (not working)
